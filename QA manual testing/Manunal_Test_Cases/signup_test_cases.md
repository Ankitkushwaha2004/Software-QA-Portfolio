## Signup Test Cases

### TC-SIGNUP-01
**Title:** Signup with valid details  

**Steps:**
1. Open signup page
2. Enter valid user details
3. Click Submit

**Expected Result:** Account should be created successfully

---

### TC-SIGNUP-02
**Title:** Signup with empty fields  

**Steps:**
1. Open signup page
2. Leave mandatory fields empty
3. Click Submit

**Expected Result:** Validation error should be shown
